# changelog · 2026-05

## 2026-05-14 — 플랫폼 운영자 사이트 IA 재구성

### 변경 내용

- IA를 객체 중심(테넌트/사용자)에서 **작업 중심 평면 메뉴**로 재구성: 대시보드 · 회원 · 문의 · 공지 · 테넌트 · 감사 로그(6개).
- 공지 모델(`Announcement`) 신설 + 운영자 콘솔 CRUD(작성·수정·발행·취소·삭제). 사용자 노출 UI는 다음 사이클.
- 신규 백엔드 엔드포인트 5종: `GET /platform/overview`, `GET /platform/users`(기본 목록), `GET /platform/inquiries`(크로스 테넌트), `Announcements` CRUD(`/platform/announcements`), `GET /platform/audit-logs`.
- `apps/api/src/routes/platform.ts` 단일 파일을 `apps/api/src/routes/platform/` 디렉터리로 분할(tenants/users/inquiries/overview/announcements/audit/lib).
- `recordAuditLog` 헬퍼 도입. 설정 갱신·문의 답변·공지 쓰기 액션을 비즈니스 쓰기와 **동일 Prisma 트랜잭션**에서 감사 로그 INSERT(PRD §4.4 A-04).
- 프론트엔드 신규 페이지: `/platform`(대시보드 KPI), `/platform/users`(기본 목록 + 검색 통합), `/platform/inquiries`(크로스 테넌트 통합 목록), `/platform/announcements`·`/new`·`/[id]`(공지 관리), `/platform/audit-logs`(필터·페이지네이션 표). 모두 페이지당 15건·다크모드 토큰·반응형.
