# 설정 UI 정적 목업 (안 A)

## 보는 방법

- **파일로 바로 열기:** `index.html`을 브라우저로 연다.
- **로컬 서버(권장):** 레포 루트에서  
  `npx --yes serve docs/design/artifacts/settings-ui-mockup -p 3456`  
  후 브라우저에서 `http://localhost:3456`

## 포함 기능

- 사이드바 네비: 종목 · 테마 · 뉴스 규칙 · **런타임 설정**
- 라이트/다크 토글(헤더 **테마** 버튼)
- 상단 라디오로 **기본 / 로딩 / 빈 / 오류** 상태 데모(종목 영역 중심)
- `globals.css` 와 맞춘 **OKLCH 토큰** 축약본
