# 안 A — 로컬 와이어 (모바일 반응형)

텍스트 기준 목업. 구현은 [`docs/requirements/mobile-dashboard-responsive.md`](../requirements/mobile-dashboard-responsive.md)와 동일 브레이크포인트를 따른다.

## 화면 1 — 모바일 목록 (필터 닫힘)

```
┌─────────────────────────────┐
│ 관심종목 모니터링    [테마][설정] │
│ WS ●  [필터]                    │
├─────────────────────────────┤
│ [종목명·코드·별칭 검색____] [초기화] │
├─────────────────────────────┤
│ 관심종목          [종목 추가]   │
│ ┌─────────────────────────┐ │
│ │ 표 (가로 스크롤)          │ │
│ └─────────────────────────┘ │
│ … 차트 또는 안내 …            │
├─────────────────────────────┤
│ 테마                          │
│ [칩] [칩] …                   │
├─────────────────────────────┤
│ 관련 뉴스                     │
│ · 기사 …                     │
└─────────────────────────────┘
```

## 화면 2 — 필터 시트 열림 (`dialog`)

```
┌─────────────────────────────┐
│ 상세 필터              [닫기] │
│ 시장    [▼ 전체    ]         │
│ 세션    [▼ 전체    ]         │
│ NXT     [▼ 전체    ]         │
│ ☑ 등락률 알림  기준 [±10%▼] │
│ (오류 메시지 영역)           │
└─────────────────────────────┘
```

## 화면 3 — 종목 선택 후 (차트 노출)

화면 1과 동일 단, 관심종목 패널 하단에 `PriceChartPanel` 영역 확장. 접기 시 선택 해제와 동일 동작 유지.

## 데스크톱 (980px 이상)

기존 3열 그리드. 헤더 한 줄에 검색·셀렉트·알림·설정 유지.
