# 뉴스 노이즈·누락 샘플 테스트 계획

**전제:** D-002에 따라 네이버 뉴스 검색(또는 확정 API) 연동 후 실행.

## 준비

- 테스트 종목 5개: 대형주 2, 중소형 1, 별칭 많은 종목 1, 동명이의 가능 1
- 각 종목에 `searchAlias`를 실제 운영과 비슷하게 설정

## 케이스

1. **종목명만** 검색했을 때 상위 20건 중 관련 기사 비율(주관 판단) 기록
2. **별칭만** 검색했을 때 동일
3. **GLOBAL exclude** 규칙(예: `광고`) 적용 후 건수 변화
4. **STOCK include** 규칙 적용 후 건수·품질 변화

## 합격 기준 (초안)

- 관련 없는 기사가 상위 10건 중 과반수면 → 검색어 조합·규칙 조정 이슈로 등록
- 뉴스 0건이 연속이면 → API 한도·쿼리·장애 로그 확인

## 산출물

- 스프레드시트 또는 이슈 코멘트에 표 1개 + 스크린샷(선택)
